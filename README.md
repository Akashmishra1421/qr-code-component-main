### What I learned

This was one of my first Frontend Mentor challenges. I practiced centering a card on the page using Flexbox:

```css
body {
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}
```

I also learned how to make the card responsive using `width: 100%` with `max-width`, so it scales properly on smaller screens.
